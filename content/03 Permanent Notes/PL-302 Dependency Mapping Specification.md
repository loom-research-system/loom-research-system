# Dependency Mapping Specification

## Purpose

Dependency Mapping traces the chains of dependency that connect policy declarations to public outcomes, and assesses where critical dependencies are concentrated.

## Status

This specification is in Draft. The methodology has been demonstrated in Cases [[PL-402|#001]], [[PL-403|#002]], and [[PL-404|#003]] through Stage 3 of the [[PL-201 Loom Analysis Protocol|Analysis Protocol]] (Dependency Chain Analysis). The standalone specification will formalize:

- Methods for identifying and documenting dependency chains
- Node classification (controllable, influenceable, external)
- Dependency concentration assessment methodology
- Single point of failure identification
- Visualization conventions for dependency network diagrams

## Development Path

Full specification to be developed during Phase 2 (Systems Development).
