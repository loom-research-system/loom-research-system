# Research Standards

## Evidence Hierarchy

| Level | Description | Example |
|-------|-------------|---------|
| **Primary** | Original data, direct observation | Interview transcript, raw dataset |
| **Secondary** | Scholarly analysis of primary | Peer-reviewed article, book |
| **Tertiary** | Summary or synthesis of secondary | Literature review, textbook |
| **Grey** | Non-peer-reviewed but relevant | Policy brief, NGO report, preprint |

## Concept Maturity

Track where each idea stands in your thinking:

| Stage | Meaning | Action |
|-------|---------|--------|
| **Observation** | Noticed something interesting | Record in Sources or Literature Note |
| **Candidate** | Think there might be a pattern here | Create Permanent Note, mark as tentative |
| **Supported** | Found multiple sources confirming | Strengthen Permanent Note, add evidence |
| **Canonical** | Confident enough to build on | Use in Outputs, cite as established |

## Citation Rules

1. Every claim has a source
2. Every source has a Literature Note
3. Every Literature Note links to its Source
4. Every Permanent Note links to supporting Literature Notes

## Versioning for Outputs

| Stage | Meaning | Who Sees It |
|-------|---------|-------------|
| **Draft** | Rough, incomplete | Only you |
| **Working** | Complete but unreviewed | Collaborators |
| **Stable** | Reviewed and approved | Wider audience |
| **Archived** | Superseded by newer version | Reference only |

## Red Flags

- A Permanent Note with no links to Literature Notes
- A Source with no Literature Note
- A Literature Note that only summarizes without analyzing
- An Output with no connection to Permanent Notes

## Quality Checklist

Before marking any note as "done":

- [ ] Written in my own words (not copied)
- [ ] Links to at least one other note
- [ ] Has complete metadata (type, id, created, tags)
- [ ] Evidence is cited (for Permanent Notes)
- [ ] Critique is included (for Literature Notes)
