# Pattern Reports

Draft. No Pattern Reports have been generated yet. The [[PL-501 Cross-Case Comparison Matrix|Cross-Case Matrix]] currently contains 3 cases. Pattern Reports require patterns to persist across 5+ cases with diverse configurations.

## Template

Each Pattern Report will follow this structure:

### Pattern Report #XXX: [Pattern Name]

**Observation:** What pattern has been observed?

**Evidence:** Which cases demonstrate this pattern?

**Possible Mechanism:** What might explain this pattern?

**Limitations:** What conditions might limit the pattern's applicability?

**Testable Hypothesis:** What prediction does this pattern generate for future cases?
